# 🎙️ Voice AI Assistant - DIO Challenge

## 📌 Descrição
Projeto desenvolvido como desafio da DIO para construção de um assistente de voz utilizando:

- Speech-to-Text (Whisper)
- Modelo de linguagem (Transformers)
- Text-to-Speech (SpeechSynthesis API)

## 🚀 Funcionalidades
✔ Upload de áudio  
✔ Transcrição automática  
✔ Geração de resposta com IA  
✔ Reprodução por voz  

## 🛠 Tecnologias
- Python
- OpenAI Whisper (modelo local)
- Hugging Face Transformers
- Google Colab

## 🎯 Objetivo
Demonstrar integração entre processamento de áudio e inteligência artificial.

---
Projeto desenvolvido para fins educacionais.
